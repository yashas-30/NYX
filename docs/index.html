◆ Core Features

Everything you need.
Nothing you don't.

NYX is engineered for developers who want full control — over models, hardware, and output quality.

🖥️
Local GGUF Runner
Built-in llama-server binary. Download, load, and run GGUF models on your GPU — no Ollama or LM Studio required.
⚙️
Per-Model Inference Controls
GPU Layers, Context Size, CPU Threads, Temperature, Top-P/K, Mirostat — per model, resets on model switch.
🤖
NYX Agent Pipeline
3-stage Architect → Coder → Optimizer pipeline. Works with any selected model. Only the final polished output shown.
⚡
Zero-Delay Streaming
Dual Express + Fastify architecture. TCP Nagle disabled, DNS pre-warmed, SHA-256 disk cache for 0ms repeat queries.
📚
Codebase Knowledge
Index your project files and query them contextually during generation. Codebase-aware code suggestions.
🔐
100% Local Keys
API keys stay in your browser's localStorage. Never stored in a database. Never sent to third parties.

◆ NYX Agent

Three-stage intelligence.
One perfect output.

Every prompt runs through a sequential pipeline of specialized agents — using whichever model you select.

Stage 01
Architect
Designs the system blueprint, breaks the problem into components, and produces a technical specification.
Stage 02
Coder
Implements the full solution from the blueprint. Writes complete, production-ready code — never truncated.
Stage 03
Optimizer
Audits, refines, and delivers the final polished output. This is the only stage you see.
